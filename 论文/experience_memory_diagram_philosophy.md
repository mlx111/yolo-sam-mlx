# Layered Evidence

The visual language treats robotic memory as an evidence system: records are not decorative cards but calibrated traces that either support action or prevent repetition of known failure. Space should separate the positive and negative functions of memory clearly, while still showing that both belong to one closed-loop mechanism. The composition is deliberately precise, like a technical figure refined for a conference presentation.

Color carries semantic responsibility. Validated memory uses a restrained blue-green range to suggest reusable, trusted experience; failed memory uses a muted red range to suggest caution and blocking rather than alarm. Neutral gray and warm white create a laboratory surface, allowing arrows, counts, and labels to become the main explanatory rhythm.

The figure should feel engineered rather than ornamental. Rounded modules, thin connectors, and small record tokens communicate hierarchy and data flow. Text is sparse and functional: each label names a mechanism or a measurable fact. The viewer should understand the story within seconds during a PPT explanation.

The final artifact should look meticulously crafted: balanced spacing, consistent stroke widths, carefully aligned text baselines, and enough negative space to prevent visual congestion. The design should support oral explanation rather than replace it with paragraphs.
